Exercícios:

1 - Desenvolva um algoritmo que receba dois valores e imprima no console o maior deles. Usar a estrutura de decisão "IF".

2 - Desenvolva um algoritmo para imprimir na tela se o voto do(a) usuário(a) é obrigatório, sabendo que a idade dele(a) é 18 anos. Usar apenas o IF.

3 - Desenvolva um programa para calcular a media de um(a) estudante com base em duas notas. Se a nota for maior ou igual a 7, devera apresentar a mensagem"Voce esta aprovado com a media"(Inserir a média calculada).Caso contrário, deverá apresentar a mensagem: "Voce está reprovado(a) com a media (inserir a media calculada).Voce precisa estudar mais"
